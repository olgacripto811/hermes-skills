# Движение капитала инвестора

## Правило
ОБЯЗАТЕЛЬНО вызывай ~/capital_add.sh через инструмент terminal.
НИКОГДА не имитируй. НИКОГДА не выдумывай результат.
Один вызов скрипта — одна реальная запись в Notion.

## Шаги

### 1. Извлеки из сообщения пользователя:
- TYPE: «Пополнение» или «Вывод»
- AMOUNT: сумма числом (только цифры, без знаков)
- INVESTOR: имя (если не указано — оставь пустым)
- DATE: дата в формате YYYY-MM-DD (если не указана — сегодня)
- NOTE: заметка (если есть)

### 2. Выполни через terminal одну команду:

```bash
~/capital_add.sh "TYPE" AMOUNT "INVESTOR" "DATE" "NOTE"
```

Примеры:
```bash
~/capital_add.sh "Пополнение" 1000 "Иван" "2026-06-27" ""
~/capital_add.sh "Вывод" 500 "Иван" "2026-06-27" "частичный вывод"
~/capital_add.sh "Пополнение" 2000 "" "" ""
```

### 3. Покажи пользователю вывод скрипта дословно.

## Важно
- Если пользователь не указал дату — передай пустую строку "" и скрипт возьмёт сегодня
- Сумма всегда положительная — знак определяет TYPE
- Не добавляй лишних действий после записи
